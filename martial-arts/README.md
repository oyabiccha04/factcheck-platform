# 格闘技ログ（Martial Arts Logs）

本ディレクトリは、factcheck-platform における  
**格闘技ジャンルの入口および方針定義**を管理するためのものです。

## 対象期間
- 2020年以降

## 対象団体・競技
- RIZIN
- ボクシング世界戦
- ONE Championship
- K-1
- UFC（2025年以降の日本人挑戦）

## 対象興行
- アリーナクラス
- ドームクラス

## 収録方針
- 注目度の高い対戦カードのみを記録対象とする
- 全カード・全大会の網羅は行わない
- 予測・発言・結果を分離して記録する
- 評価・優劣・ランキングは行わない

## UFC日本人挑戦の扱い（2025年以降）
- 世界ランキング・タイトル戦線に関与する試合のみを対象とする
- 日本開催・海外開催は問わない
- 単なる参戦ではなく、「挑戦」と言える文脈を持つカードに限定する

## 目的
まずは格闘技ジャンルにおいて、  
**「ログとして十分な密度を持つ状態」**  
を作ることを最優先とする。
