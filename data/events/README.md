# factcheck-platform

## 概要
factcheck-platform は  
**発言・予測そのものを「検証可能なログ」として保存するためのアーカイブサイト**です。

評価・意見・正誤判定は一切行いません。  
「誰が・いつ・何を言い、結果として何が起きたか」を  
**第三者が後から検証できる形で残す**ことだけを目的とします。

本プロジェクトでは  
**GitHub の commit 履歴そのものも検証対象**とみなします。

---

## 基本思想（絶対に崩さない前提）

- 主語は「人」ではなく **発言（statement）**
- 発言内容と、後から判明した結果（verification）は分離
- 後出し修正は **commit 履歴で必ず可視化**
- JSON は **差分更新禁止／毎回全文更新**
- **1イベント = 1 JSON**
- 静的配信（GitHub Pages）前提
- MVP 段階では評価ロジック・スコアリングを入れない
