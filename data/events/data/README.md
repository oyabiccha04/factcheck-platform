# Data Directory

このディレクトリは、本プロジェクトにおける
**検証可能な一次データのみ**を格納する。

## 構成方針

- データは表示や評価のために加工しない
- 意味解釈・集計・表示ロジックは含めない
- JSONは「その時点で存在した事実」を固定する

## ディレクトリ構成

- `events/`
  - 公開検証の最小単位
  - 1イベント = 1JSON
  - イベント一覧は `events/index.json` にのみ定義する

## 禁止事項

- 後出しで意味を書き換える修正
- 評価語・断定的結論の追記
- 既存JSONへの差分的更新

このディレクトリ配下のデータは、
GitHubの commit 履歴と合わせて検証対象となる。
