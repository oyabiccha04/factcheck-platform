About

このサイトは、格闘技・ボクシングの試合や興行を中心に、世に出た「予測」と、あとから確定した「事実（結果）」を分けて記録し、誰でも検証できる形で公開するアーカイブです。

このサイトがねらっていること

一言でいうと、「予測を、ファクトで評価し、必要に応じて的中率などで可視化する」ことです。

  • 予測は、いつ・誰が・どんな根拠で・何を言ったかを、可能な限り一次情報にたどれる形で残す。
  • 結果が出たら、予測と結果を混ぜずに照合し、その履歴を残す。
  • 世の中には予測があふれていますが、結果と照合されないまま消えていくものも多い、という偏りに対し、「事実としての履歴がある」状態をつくることを重視します。
  • いまは格闘技・ボクシングを主な題材にしていますが、同じ枠組みは他分野にも広げうる実証の場という位置づけでもあります。

大切にしているスタンス（やらないこと）

  • 誰かを断罪したり、格付けしたり、正しさを裁定することが目的ではありません。ランキングや的中率は、あくまで照合結果を数値化した事実指標の提示にとどめ、煽り・誹謗・人格攻撃を目的とした表現や設計にはしません。
  • 編集部の意見や「当たり外れの総評」を主コンテンツにはしません。判断は閲覧者に委ね、ここにはログと照合の記録を置くことを優先します。
  • 感情的な支持・不支持の表明や、思想的な断罪を主目的とするコンテンツにはしません。

記録と運用のルール（要約）

  • 予測と結果は分離する。後から書き換えてごまかせないよう、データの変更は Git の commit 履歴として残せる運用（GitHub 上のリポジトリ）を前提にしています。
  • 予測の主たる対象は「勝つ側（勝者方向）」です。KO・判定・サブミッションなどは、主予測を補う情報として扱います。
  • 出典は可能な限り一次情報（公式・本人発信・原記事など）を優先し、不明な場合は「不明」と明示します。推測で埋めません。
  • 誰の予測を載せるかには運用基準があります（例：当事者性・専門性の高い関係者、一定規模以上の格闘技系発信者など）。根拠が曖昧な場合は断定せず、要確認として残す方針です。
  • 集計やランキングの表示は、原則として興行・試合の結果が確定したうえで扱い、途中経過だけで数字を煽るような設計にはしません。
  • 同一の興行に複数種目（例：MMA とキック）が含まれる場合、種目ごとの画面ではその種目の試合に紐づく予測だけを集計の対象にします（データ上の種目情報に基づく）。

収益について（方針）

将来的に広告や寄付を導入する可能性はありますが、予測や結果の改変、特定の結論への誘導、スポンサー都合での歪曲を正当化する理由にはしません。収益は運営・データ整備・履歴保存の手段と位置づけ、思想そのものとは切り離します。

より詳しいルール・開発者向け

運用上の細目・データ形式・ページ構成の正本は、リポジトリ内の憲法ドキュメント PROJECT_CHARTER.md にまとめています。生の JSON や GitHub への直リンクは 検証用データ・リポジトリ からどうぞ。データの第三者利用・免責は ポリシー・利用条件 に記載しています。
